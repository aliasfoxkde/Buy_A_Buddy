# 🎮 BUY A BUDDY - Complete RPG Design Document

> **"The Shadow King has scattered the Golden Buddies across Buddyaria. Your quest: Solve the ancient puzzles, befriend the mystical creatures, and restore harmony to the realm."**

---

## 🎯 VISION & CORE PILLARS

### 1. Story-Driven Adventure
- **Main Quest**: Rescue the 7 Golden Buddies from the Shadow King
- **Side Quests**: Help villagers, solve mysteries, discover secrets
- **Puzzles**: Ancient riddles, pattern matching, element combining

### 2. Buddy Collection & Evolution
- **Catch**: Find and befriend wild buddies
- **Train**: Level up, evolve, master abilities
- **Combine**: Mix element types for rare offspring

### 3. Strategic Combat
- **Turn-based battles** with elemental weaknesses
- **Puzzle mechanics** in combat (match symbols, solve patterns)
- **Boss encounters** with unique puzzle mechanics

### 4. Puzzle Dungeons
- **Crystal Caves**: Pattern-matching puzzles
- **Shadow Temple**: Memory puzzles
- **Fire Mountains**: Timing puzzles
- **Angel Peaks**: Logic riddles

## 📖 STORY OVERVIEW

### Prologue
Long ago, the Golden Buddies ruled Buddyaria in peace. They were pure beings of light, protecting the realm from darkness.

The Shadow King grew jealous of their power. Using the ancient **Gem of Shadows**, he scattered the Golden Buddies across the land, hiding them in puzzle-protected prisons.

Only the **Buddy Trainer** - you - can solve the ancient puzzles, befriend the mystical buddies, and restore the Golden ones.

### Chapters

#### Chapter 1: Meadow Valley (Tutorial)
- Meet Elder Maya, learn the basics
- Catch your first Buddy (Slime)
- Solve the **Symbol Puzzle** to free the first Golden Buddy
- Boss: **Moss Golem** (simple pattern matching)

#### Chapter 2: Crystal Caves
- Discover the Cave of Reflections (mirror puzzles)
- Meet the Crystal Keeper (gives cryptic clues)
- Find the **Prism Key** by solving color patterns
- Boss: **Prism Guardian** (color mixing puzzles)

#### Chapter 3: Shadow Temple
- Navigate the dark temple (memory puzzles)
- Find the **Shadow Shard** to reveal hidden paths
- Boss: **Shadow Wolf** (light vs dark mechanics)

#### Chapter 4: Fire Mountains
- Cross the volcanic region (timing puzzles)
- Meet Blazefire, the temperamental fire buddy
- Obtain the **Flame Stone** to cool the lava
- Boss: **Magma Dragon** (elemental reaction puzzles)

#### Chapter 5: Angel Peaks
- Scale the sky mountains (air puzzles)
- Restore the broken bridge with wind patterns
- Obtain the **Feather of Light**
- Boss: **Storm Angel** (weather puzzle)

#### Chapter 6: Shadow King's Castle
- Final puzzle gauntlet
- Rescue all 7 Golden Buddies
- Boss: **The Shadow King** (combined puzzle types)

## 🧩 PUZZLE MECHANICS

### 1. Symbol Match Puzzle
```
Pattern: Players must match symbols in correct sequence
Example: 🔴🔵🔵🟢 → 🔴🔵🔵🟢
Mechanic: Memory + speed
Location: Tutorial, Shadow Temple
```

### 2. Element Combine Puzzle
```
Pattern: Combine elements to create effects
Example: 🔥 + 💧 = 🌫️ (steam)
Mechanic: Elemental reactions
Location: Crystal Caves, Fire Mountains
```

### 3. Pattern Sequence Puzzle
```
Pattern: Complete the sequence
Example: ⬛⬜⬛⬜⬛? → ⬜
Mechanic: Logic deduction
Location: Angel Peaks, Quest Tower
```

### 4. Memory Match Puzzle
```
Pattern: Find matching pairs
Example: 8 pairs hidden, flip two at a time
Mechanic: Memory
Location: Shadow Temple
```

### 5. Timing Puzzle
```
Pattern: Activate at the right moment
Example: Hit the target when the ring is full
Mechanic: Rhythm & timing
Location: Fire Mountains, Boss battles
```

## ⚔️ COMBAT SYSTEM

### Turn-Based Battle Structure
```
1. Player Turn:
   - Choose Action (Attack/Defend/Skill/Item/Flee)
   - Select Target
   - Execute Move
   
2. Enemy Turn:
   - AI selects action
   - Execute Move
   
3. Check Victory/Defeat
```

### Element Types & Weaknesses
```
Fire     → Weak vs Water   | Strong vs Nature
Water    → Weak vs Electric | Strong vs Fire  
Nature   → Weak vs Fire    | Strong vs Water
Electric → Weak vs Nature  | Strong vs Water
Dark     → Weak vs Light   | Strong vs Angel
Light    → Weak vs Dark    | Strong vs Shadow
```

### Battle Puzzles

#### Puzzle Attack System
During combat, special "Puzzle Moments" occur:

1. **Symbol Chain**: Connect matching symbols for combo damage
2. **Element Mix**: Combine elements for elemental reactions
3. **Quick Math**: Solve equation for critical hit
4. **Memory Match**: Remember sequence for bonus attack

### Special Moves
```
Quick Time Events (QTE):
- When QTE appears, press correct key in time
- Success: Critical hit + special animation
- Failure: Miss turn but no damage taken

Combo System:
- Chain attacks with matching elements
- 3-chain: 1.5x damage
- 5-chain: 2x damage + special effect
```

## 🧩 PUZZLE DUNGEON DESIGN

### Crystal Caves - "The Prismatic Trial"

#### Room 1: Color Mixer
```
Puzzle: Mix colored lights to create the correct color

Solution:
- Three light sources: Red, Blue, Yellow
- Target: Create Purple (Red + Blue)
- Mechanism: Drag lights to prism, see result

Reward: Shimmerpuff encounter
```

#### Room 2: Pattern Lock
```
Puzzle: Complete the crystal pattern

Solution:
- 3x3 grid of crystals
- Some lit, some dark
- Pattern: Odd positions lit, even dark
- Press crystals to toggle

Reward: Crystal Shard key item
```

#### Room 3: Prism Maze
```
Puzzle: Guide light through mirrors

Solution:
- Place mirrors to reflect light
- Light must reach the exit prism
- Limited mirrors per room

Reward: Access to Prism Guardian boss
```

### Shadow Temple - "The Memory Challenge"

#### Room 1: Torch Sequence
```
Puzzle: Light torches in correct order

Solution:
- 5 torches numbered
- NPC gives sequence: 3-1-4-1-5
- Light in order within time limit

Reward: Shadow Shard
```

#### Room 2: Rune Memory
```
Puzzle: Match ancient runes

Solution:
- 8 rune pairs hidden face-down
- Flip two at a time
- Match all to proceed

Reward: Shadow Puff (buddy)
```

#### Room 3: Echo Chamber
```
Puzzle: Repeat the shadow sounds

Solution:
- Listen to sequence of shadow sounds
- Repeat via clicking shadow orbs
- Length increases per round

Reward: Access to Shadow Wolf boss
```

### Fire Mountains - "The Volcano Trial"

#### Room 1: Lava Hop
```
Puzzle: Cross lava by activating safe platforms

Solution:
- Platforms appear/disappear in pattern
- Step on correct platform when lit
- Wrong step = restart room

Reward: Fire Gem
```

#### Room 2: Element Chain
```
Puzzle: Chain elements for correct reaction

Solution:
- Elements: Fire, Water, Nature, Electric
- Chain: Fire + Water = Steam (safe)
- Chain: Fire + Nature = Explosion (damage)
- Must create Steam paths to progress

Reward: Blazefire encounter
```

#### Room 3: Timing Gems
```
Puzzle: Hit gems when timing ring is perfect

Solution:
- Gems float across screen
- Timing ring appears
- Hit when ring is full
- 5 successful hits = reward

Reward: Flame Stone key item
```

### Angel Peaks - "The Sky Trial"

#### Room 1: Cloud Bridge
```
Puzzle: Balance on clouds by matching symbols

Solution:
- Clouds have symbols (+ or -)
- Standing on cloud affects balance meter
- Must reach 0 to cross

Reward: Wind Gem
```

#### Room 2: Constellation Match
```
Puzzle: Connect stars to form constellations

Solution:
- Stars scattered on sky map
- Connect in correct pattern
- Forms: Bear, Swan, Dragon
- Each gives a power-up

Reward: Angel Feather key item
```

#### Room 3: Wind Tunnel
```
Puzzle: Use wind currents to navigate

Solution:
- Wind arrows show direction
- Jump between wind streams
- Reach the top without falling

Reward: Cloudangel encounter
```

## 🎮 GAME MECHANICS

### Buddy Collection
```
Catch Method:
1. Weaken wild buddy to low HP
2. Use Buddy Ball item
3. Higher catch rate with:
   - Lower enemy HP
   - Status effects (sleep, paralysis)
   - Rarer buddy = lower catch rate

Buddy Types:
- Wild: Random stats, need training
- Bred: Inherit parent traits
- Found: Secret puzzles, special abilities
- Golden: Legendary, one per chapter
```

### Leveling & Evolution
```
Experience:
- Win battles: 10-50 XP
- Complete quests: 100-500 XP
- Find secrets: 200 XP

Level System:
- Level 1-10: +5 to all stats per level
- Level 11-20: +8 to all stats per level
- Level 21-30: +12 to all stats per level
- Level 31+: +15 to all stats per level

Evolution Requirements:
- Reach level threshold
- Use Evolution Item
- Sometimes puzzle to unlock
```

### Idle Farm System
```
Work Plots:
- 6 plots available
- Assign buddies to work
- Earn coins per second based on:
   - Buddy base income
   - Buddy level
   - Plot upgrade level

Upgrades:
- Plot Level: +25% income per level
- Buddy Training: +10% income per level
- Auto-Collector: Offline earnings
```

### Item System
```
Consumables:
- Buddy Ball: Catch wild buddies
- Potion: Restore 50 HP
- Super Potion: Restore 150 HP
- Revive: Restore fainted buddy
- Lucky Charm: +20% catch rate

Key Items:
- Water Stone: Evolve water buddies
- Fire Stone: Evolve fire buddies
- Crystal Shard: Open crystal doors
- Shadow Shard: Reveal hidden paths
- Angel Feather: Unlock angel areas
- Golden Dust: Evolve to legendary
```

### Quest System
```
Main Quests (Story):
- Chapter progression
- Boss battles
- Key item collection

Side Quests:
- Help NPCs
- Collect items
- Battle challenges
- Puzzle mastery

Daily Quests:
- Catch 3 buddies
- Win 5 battles
- Earn 1000 coins
- Solve 2 puzzles
```

📋 How to Use These Templates

  1. Download the template images you need
  2. Fill each grid section with your custom artwork
  3. Save the completed spritesheet as PNG
  4. Place in public/images/sprites/
  5. Use CSS background-position to select sprites

📐 CSS Sprite Positioning

Character/Buddy Spritesheet (6×2 grid)

.sprite {
  width: 256px;
  height: 512px;
  background-image: url('/images/sprites/characters.png');
  background-repeat: no-repeat;
}

/* Row 1 */
.char-1-1 { background-position: 0 0; }
.char-1-2 { background-position: -256px 0; }
.char-1-3 { background-position: -512px 0; }
.char-1-4 { background-position: -768px 0; }
.char-1-5 { background-position: -1024px 0; }
.char-1-6 { background-position: -1280px 0; }

/* Row 2 */
.char-2-1 { background-position: 0 -512px; }
.char-2-2 { background-position: -256px -512px; }
.char-2-3 { background-position: -512px -512px; }
.char-2-4 { background-position: -768px -512px; }
.char-2-5 { background-position: -1024px -512px; }
.char-2-6 { background-position: -1280px -512px; }
